# Weather Data Collector (Python)

This project collects real-time weather data (temperature & humidity) using the Open-Meteo API.
The script:
- Converts city name → coordinates
- Calls the weather API
- Extracts temperature & humidity
- Saves data to CSV
## Output Sample
2025-12-10 | Cairo | 22.5°C | 48%
## How to run

pip install -r requirements.txt
python main.py
